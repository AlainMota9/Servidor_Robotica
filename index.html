Aplicativos e Notas

Arduino Language Reference é um aplicativo que fornece todas informações sobre a linguagem de programação para Arduino.

Electronic Circuit é um aplicativo sobre eletrônica que traz os mais diversos esquemas elétricos.

Electronics Toolkit é um aplicativo sobre eletrônica focado em códigos de resistores, capacitos e indutores, trazendo as mais diversas informações dos componentes.

Arduino Droid é um aplicativo que funciona como IDE (ambiente de desenvolvimento integrado) fornecendo todas as ferramentas necessárias para programar o Arduino.

As Notas de Aulas são a coleção de todos os Slides das aulas de robótica anterioes.
